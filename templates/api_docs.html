REST API

Доступ по сессии (нужно быть авторизованным). Базовый URL: {{ request.url_root }}api

МетодURLОписание
GET/api/listsСписок всех списков
POST/api/listsСоздать список (JSON: {"name": "..."})
GET/api/lists/<id>Получить список с товарами
POST/api/lists/<id>/itemsДобавить товар (JSON: {"name": "...", "quantity": 1, "category_id": 1})
POST/api/lists/<id>/items/<item_id>/toggleОтметить купленным/не купленным
DELETE/api/lists/<id>/items/<item_id>Удалить товар
GET/api/templatesСписок шаблонов

Пример: curl -X POST -H "Content-Type: application/json" -d '{"name":"Тест"}' {{ request.url_root }}api/lists (с cookies сессии)
